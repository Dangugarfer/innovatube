# Frontend - InnovaTube

Este proyecto corresponde al frontend de **InnovaTube** y fue desarrollado con **Angular 18** y **Angular Material**.

## Tecnologías utilizadas

* Angular 18
* Angular Material
* TypeScript
* RxJS

## Notas

Este frontend consume la API desarrollada en Node.js, por lo que es necesario que el backend esté en ejecución para utilizar todas las funcionalidades de la aplicación en un entorno local.

### Despliegue

**Frontend (Vercel):**
https://innovatube-phi.vercel.app/

**Backend (Render):**
https://innovatube-iqop.onrender.com/

> **Importante:** El backend está desplegado en el plan gratuito de Render. Si la aplicación no ha recibido solicitudes durante un tiempo, el primer acceso puede tardar algunos segundos mientras el servicio vuelve a iniciarse. Una vez activo, la aplicación funcionará con normalidad.
